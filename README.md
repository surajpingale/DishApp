# Dish App

DishApp allows to user to save dish recipe in local database and fetch random recipe online.


## Technologies

* Kotlin
* ViewBinding
* Room
* Retrofit
* Coroutines
* Glide
* Recycler View
* MVVM
* Live Data
* Navigation Component
* Diff Utils
* RxJava
* Notifications
* Work Manager
* Broadcast Listener
